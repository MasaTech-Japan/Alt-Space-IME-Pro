■ 概要
「Alt-Space IME Pro」は、日本語入力の切り替えを劇的にスムーズにするツールです。
Macのように左右のAltキー単体でIMEのオン・オフを切り替えられるほか、
Spaceキーの長押しや変換・無変換キーへの機能割り当てなど、
あなたのタイピングスタイルに合わせた最適な入力環境を提供します。

■ 主な機能
1. 左右Altキーの単体押しでIMEをオン・オフ切り替え（空打ち対策済み）。
2. Spaceキーの長押し（0.3秒以上）によるIME切り替え。
3. 変換・無変換キーによるIME切り替え（設定でON/OFF可能）。
4. 設定画面（Ctrl+Alt+I）から、自分好みのスタイルにカスタマイズ。
5. スタートアップ登録機能により、Windows起動時に自動実行。

■ 使用方法
1. AltSpaceIME.exe を実行してください。又はAltSpaceIMEPro_v2,0.zipを解凍してAltSpaceIMEPro_v2,0.exeを実行して下さい。
2. タスクトレイにアイコンが表示されます。
3. デフォルトでは「左右Altキー」または「Space長押し」でIMEが切り替わります。
4. 設定を変更したい場合は、トレイアイコンを右クリックして「設定」を開くか、
   ショートカット [Ctrl + Alt + I] を押してください。

■ ショートカットキー
・[Ctrl + Alt + I]：設定画面を表示
・[Ctrl + Alt + R]：IME状態のリセット
・[Ctrl + Alt + Q]：アプリの終了

■ インストール・アンインストール
・インストール：適当なフォルダに解凍して実行してください。
・アンインストール：実行ファイルを削除してください。
  ※設定は同フォルダの AltSpaceIME.ini に保存されます。
