### ■ 概要
「Alt-Space IME Pro」は、日本語入力の切り替えを劇的にスムーズにするツールです。
Macのように左右のAltキー単体でIMEのオン・オフを切り替えられる他、Spaceキー最初の押しや変換・無変換キーへの機能割り当てなど、あなたのタイピングスタイルに合わせた最適な入力環境を提供します。

### ■ インストール・アンインストール
* **インストール**：適当なフォルダに解凍して実行してください。
* **アンインストール**：実行ファイルを削除してください。
※設定は同フォルダの AltSpaceIME.ini に保存されます。

### ■ 主な機能
* 左右Altキーの単体押しでIMEをオン・オフ切り替え（空打ち対策済み）。
* スペースキーの長押し（0.3秒以上）によるIME切り替え。
* 変換・無変換キーによるIME切り替え（設定でON/OFF可能）。
* 設定画面（Ctrl+Alt+I）から、自分好みのスタイルにカスタマイズ。
* スタートアップ登録機能により、Windows起動時に自動実行。

### ■ 使用方法
1. AltSpaceIMEPro_v2,3.0.zip を解凍して AltSpaceIMEPro_v2,3.0.exe を実行してください。
2. タスクトレイにアイコンが表示されます。
3. 基本操作では、「左右Altキー」または「Space長押し」でIMEが切り替わります。
4. 設定を変更したい場合は、トレイアイコンを右クリックして「設定」を開くか、ショートカット [Ctrl + Alt + I] を押してください。

### ■ ショートカットキー
* [Ctrl + Alt + I]：設定画面を表示
* [Ctrl + Alt + R]：IME状態のリセット
* [Ctrl + Alt + Q]：アプリの終了

### ■ IME状態が不安定になった時の復旧方法
もしIMEの状態表示が「_ア」や「_A」になり、通常の切り替えができなくなった場合は、以下のキーを試してください。

#### 「_ア（カタカナ）」状態の場合
* 「カナ/かな」キー、または「無変換」キーを押すことで復旧できます。（※変換・無変換キー設定が「無効」の場合）

#### 「_A（全角英数）」状態の場合
* 「カナ/かな」キーを押すことで復旧できます。

> ※これはシステムの仕様によるものなので、Alt-Space IME Proを使用していない時でも、起こる動作です。
