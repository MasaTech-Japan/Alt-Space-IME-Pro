# Alt-Space IME Pro v1.8

**「不便」を「便利」に。日本語入力を劇的にスムーズにするプロ仕様ツール。**

## 👤 作成者
- [cite_start]**まさゆき (Masayuki)** 
- [cite_start]**動作環境**: Windows 10 / 11 (AutoHotkey本体のインストールは不要です) 

## 📝 概要
[cite_start]「Alt-Space IME Pro」は、日本語入力の切り替えをスムーズにするツールです 。
[cite_start]Macのように左右のAltキー単体でIMEのオン・オフを切り替えられるほか、Spaceキーの長押しや変換・無変換キーへの機能割り当てなど、あなたのタイピングスタイルに合わせた最適な入力環境を提供します 。

## ✨ 主な機能
1. [cite_start]**左右Altキーの単体押し**: IMEをオン・オフ切り替え（空打ち対策済み） 。
2. [cite_start]**Spaceキーの長押し (0.3秒以上)**: IME切り替え 。
3. [cite_start]**変換・無変換キー**: IME切り替え（設定でON/OFF可能） 。
4. [cite_start]**カスタマイズ**: 設定画面（`Ctrl+Alt+I`）から、自分好みのスタイルに調整可能 。
5. [cite_start]**自動実行**: スタートアップ登録機能により、Windows起動時に自動で立ち上がります 。

## ⌨️ ショートカットキー
- [cite_start]`Ctrl + Alt + I` : 設定画面を表示 
- [cite_start]`Ctrl + Alt + R` : IME状態のリセット 
- [cite_start]`Ctrl + Alt + Q` : アプリの終了 

## 🚀 使用方法
1. [Releases](https://github.com/MasaTech-Japan/Alt-Space-IME-Pro/releases) から最新の `AltSpaceIME_v1.8.exe` をダウンロードしてください。
2. [cite_start]実行するとタスクトレイにアイコンが表示され、すぐに使えます 。
3. [cite_start]設定を変更したい場合は、トレイアイコンを右クリックするか、`Ctrl + Alt + I` を押してください 。
